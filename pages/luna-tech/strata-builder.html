📖 암석 레시피

🌑 현무암
🌋 용암 + 💧 물 (급냉!) → 빠르게 식으면 현무암
⬜ 화강암
🌋 용암을 붓고 그냥 기다리기 (서서히 냉각) → 화강암
🖤 흑요석
🌋 뜨거운 용암 위에 💧 물 직접 부으면 순간 냉각 → 흑요석
🟫 사암
규소 모래를 높이 쌓고 ⏳ 기다리기 → 압력으로 굳어서 사암
🤍 석회암
석회 모래를 높이 쌓고 ⏳ 기다리기 → 석회암
🩶 셰일
점토를 높이 쌓고 ⏳ 기다리기 → 셰일
💎 대리석
🤍 석회암 만들고 → 그 위에 🌋 용암 부으면 → 열변성 → 대리석!
🪨 편암
🩶 셰일 만들고 → 그 위에 🌋 용암 부으면 → 열+압력 → 편암!
🆕 새 재료
🧊 얼음 — 온도가 오르면 물로 녹음
🛢️ 기름 — 물 위에 뜸! 300°C 이상이면 불탐
🧪 산성 — 암석과 모래를 녹임 (단열벽은 안 녹음)
⚙️ 금속 — 열전도 3배! 1500°C에서 녹음
🌍 지각 힘 (화살표 드래그)
← 왼쪽 힘 — 버튼 클릭 → 캔버스 드래그(방향+세기)
→ 오른쪽 힘 — 버튼 클릭 → 캔버스 드래그(방향+세기)
▶ 실행 — 3초간 힘이 모든 입자를 밀어냄!
드래그 길이 = 힘의 크기, 가까울수록 강한 힘
🌳 생태계 시스템
🌱 씨앗 — 흙 위에 심고 물 주면 새싹 → 나무로 성장
🔥 불 — 나무/새싹/기름을 태움, 나무→숯(탄소)
🌧️ 비 — 3초간 하늘에서 물이 내림
💎 다이아몬드 — 숯 + 고온(900°C+) + 고압(80+)!
🐾 동물 — 나무 근처에 자동 생성, 걸어다님
